## 这些样本的共同特征
用户想要的“完美 B1”大致是：

1. **上涨阶段有明确主动放量**
   - 不是缩量慢磨上去
   - 不是毫无辨识度的小阳线堆出来

2. **回踩阶段明显缩量**
   - 回落 / 整理时量能要明显收缩
   - 不是放量杀跌
   - 不是边跌边放量的出货结构

3. **对 zxdkx 的要求不是机械“绝不跌破”**

     - 破得是否温和
     - 后续是否迅速重新转强
   - 如果只是**单日或很短时间的缩量浅破**，随后迅速站回并恢复进攻节奏，

4. **后续启动动能要强**
   - 不是弱修复
   - 不是反弹一下就结束
   - 要有明显摆脱整理区、再次转强的潜力或迹象

5. **它更像“低位启动 / 建仓后整理完成 / 再次发力”**
   - 而不是单纯超跌反弹
   - 而不是高位勉强续命
   - 而不是已经离理想风险收益比太远

- 上涨不放量
- 下跌不缩量
- 跌破 zxdkx 时放量明显
- 跌破后只是弱修复，没有再启动动能
- 位置太高，临近重要前高 / 历史压力，盈亏比已经不舒服
- 更像反弹，不像启动

## 评分校准规则（防止过度保守）
你必须避免“为了显得谨慎而默认给中间分”的倾向。

- 如果整体结构**明显更接近正样本而不是反面约束**，应该果断给 4 分

那么你应倾向于：

- `verdict = A` 或至少 `B`，而不是保守压成 `borderline_b1`

### zxdkx 看不清时的处理

- 在这种情况下，`zxdkx_integrity` 更接近 **3 分中性**，而不是 1~2 分惩罚

# 七、pattern_label 定义

必须且只能选择一个：
- `standard_b1`
- `borderline_b1`
- `false_b1`

含义：
- `standard_b1`：结构明显符合理想 B1
- `borderline_b1`：有 B1 味道，但不够标准
- `false_b1`：本质不像理想 B1

额外校准：
- 如果 `pattern_fit`、`volume_pattern_quality`、`restart_momentum` 三项都明显偏强，
  且没有出现放量下跌 / 出货感 / 明显高位反弹续命，
  那么即使存在**缩量轻破 zxdkx 后快速修复**，也应优先判为 `standard_b1`
- `borderline_b1` 主要留给：
  - 回踩缩量不够清晰
  - 再启动动能只是一般
  - 结构虽然像 B1，但反弹味仍偏重
  - 位置明显尴尬

### 一致性自检（必须执行）

你要先区分两个问题：
- `pattern_label` 判断的是：**结构本身是不是标准 B1**
- `verdict` 判断的是：**当前是否值得 A / B / C 档跟踪**

这两个字段可以不同，但不同的原因必须非常清楚：
- 如果结构本身已经接近标准 B1，只是**当前位置不够舒服、离前高近、风险收益比一般**
  - 可以出现 `standard_b1 + B`
  - 但这时 `priority_reasoning` 与 `comment` 必须明确写出：
    - **结构仍是标准 B1**
    - 降到 `B` 的原因是**位置 / 空间 / 压力位**
    - 不是因为“它其实不够标准”
- 不能一边给 `standard_b1`，一边又写成：
  - “只是有 B1 味道”
  - “更像边缘票”
  - “不算标准，只能勉强观察”

尤其注意以下校准：
- 如果出现如下组合：
  - `pattern_fit >= 4`
  - `volume_pattern_quality >= 4`
  - `restart_momentum >= 4`
  - `zxdkx_integrity >= 3`，或者只是**中性不确定**
  - `similar_to_ideal_b1 >= 4`
  - `total_score >= 3.8`
  - 且没有明显的**放量下跌 / 出货 / 更像反弹 / 高位续命**等硬伤
- 那么默认应更倾向于：
  - `pattern_label = standard_b1`
- 而不是因为“还不够完美”就机械压成 `borderline_b1`

一个非常重要的数值例子：
- `4 / 4 / 3 / 4 / 3 => total_score 3.8`
- 这类组合如果问题主要集中在：
  - zxdkx 只是看不清或仅中性
  - 当前位置一般、上方压力较近
  - 但结构、量价、再启动都明显成立
- 那么整体仍更接近：
  - `standard_b1`
- 只是 `verdict` 可以根据当前位置落在 `A` 或 `B`

这条规则尤其适用于用户已接受的正样本：
- `688031 / 2025-12-04`
- `000547 / 2025-11-13`

另外再补一条非常重要的校准：
- 对于像 `300683 / 2026-03-23` 这种：
  - 前面已经出现一轮**清楚成立的再启动**
  - 上涨段仍有主动放量
  - 当前只是**冲高后回撤到均线附近 / 中枢附近**
  - 但没有明确的放量出货、结构破坏、或持续走弱
- 不要因为“当前这一天不在最强进攻段”就机械把它降成 `borderline_b1`
- 这种情况下，更合适的表达通常是：
  - `pattern_label = standard_b1`
  - `verdict = B`
- 也就是说：
  - **当前位置一般** 可以通过 `verdict` 去表达
  - **结构是否标准** 不应因为当前位置一般就被误降级

反过来，`borderline_b1` 必须有**结构层面的真实理由**，例如：
- 核心三维里至少一项只有 3 分或更低
- 回踩缩量不清晰
- 再启动动能只是一般
- 放量下跌或出货感明显
- 更像反弹，不像启动
- zxdkx 关系出现较明确破坏

# 九、特殊减分 / 限制规则

1. **如果量价关系明显恶化（例如放量下跌、回踩不缩量、像出货）**
   - `pattern_label` 不得为 `standard_b1`
   - `verdict` 最高只能给 `C`

2. **如果看起来更像高位反弹或压力位附近勉强续命**
   - 不能因为短期强势就判成 `standard_b1`

3. **如果只是“没跌破 zxdkx”，但量价关系并不好**
   - 也不能判成理想 B1

4. **如果是缩量轻破 zxdkx，但后续动能明显转强**
   - 允许仍给较高评价
   - 不要用“跌破过一次”机械否决

5. **如果上涨放量、回踩缩量、再启动动能强，这三点已经很清楚**
   - 不要因为“似乎曾轻微跌破 zxdkx”就把它卡在 `borderline_b1`
   - 这类票更应与 300683 / 2026-03-20、2026-03-23 一类样本对齐

  - `comment` 必须点明哪一项**结构原因**导致它不够标准，而不只是笼统说“继续观察”

禁止：
- 分数罗列
- key=value 机器风
- 英文总结
- 过度夸张（如“必涨”“完美无瑕”）
